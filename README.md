### 실행 방법 및 실행 시 주의 사항
- 패키지 매니저로는 `yarn`을 사용했으며, 실행 전 `yarn install`을 통해 패키지를 설치합니다.
- `post-install`을 통해 `yarn install` 이후 `cd ios && pod install && cd ..`을 적용하여 `pod install`을 실행합니다.
- `yarn start` 이 후 `iOS는 i`, `android는 a`로 시뮬레이터/에뮬레이터를 실행합니다.
